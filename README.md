# EduVibe

🚀 EduVibe — LMS Platform

EduVibe is a modern full-stack Learning Management System (LMS) designed to provide a seamless learning experience for students, instructors, and administrators.

📌 Overview

EduVibe is built as a scalable platform where:

🎓 Students can browse and learn courses
🧑‍🏫 Instructors can create and manage content
🛡️ Admins can control the system

Inspired by platforms like Udemy and Coursera.

✨ Features
🔐 JWT Authentication & Role-Based Access
📚 Course creation and enrollment
🧠 Structured learning (Modules → Topics)
📊 Role-based dashboards
🎨 Responsive UI/UX
🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
